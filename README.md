# 📱 Em Hieu Bio Link

Trang Bio Link cá nhân với hiệu ứng bong bóng Bootstrap và animation đẹp mắt.

## 📁 Cấu trúc thư mục

```
emhieu-bio/
├── index.html          (File HTML chính)
├── style.css           (File CSS với hiệu ứng)
├── README.md           (File hướng dẫn này)
└── images/
    ├── personal_photo.jpg    (Ảnh cá nhân - thay bằng file của bạn)
    └── zalo_qr.jpg           (QR code Zalo - thay bằng file của bạn)
```

## 🎯 Các bước setup

### Bước 1: Chuẩn bị ảnh
1. Tìm ảnh cá nhân của bạn (ví dụ: `profile.jpg`)
2. Đặt vào thư mục `images/` với tên `personal_photo.jpg`
3. Chuẩn bị ảnh QR Zalo (có thể dùng các app như QR Code Generator)
4. Đặt vào thư mục `images/` với tên `zalo_qr.jpg`

### Bước 2: Tùy chỉnh thông tin
Mở file `index.html` và:
- Sửa tên "Trọng Hiếu" thành tên của bạn (dòng ~31)
- Sửa mô tả "Web Developer | Cloud Enthusiast" thành của bạn
- Các link Facebook, Instagram, LinkedIn đã có sẵn

### Bước 3: Kiểm tra
Mở file `index.html` bằng trình duyệt để xem kết quả.

## ✨ Tính năng

- ✅ Hiệu ứng bong bóng floating background
- ✅ Bubble effect on button hover (Bootstrap)
- ✅ Animation mượt mà khi load trang
- ✅ Responsive design (mobile-friendly)
- ✅ Dark mode friendly (gradient background)
- ✅ Social media icons (Font Awesome)
- ✅ Hover effects với transform & shadow

## 🎨 Các hiệu ứng chính

1. **Floating Bubbles** - Bong bóng nổi ở background
2. **Bubble Effect on Click** - Hiệu ứng sóng khi hover nút
3. **Profile Photo Float** - Ảnh cá nhân lơ lửng nhẹ nhàng
4. **Scale on Hover** - Nút phóng to khi hover
5. **QR Image Zoom** - QR code phóng to khi hover

## 🚀 Deploy lên GitHub Pages

1. Tạo repository `emhieu-bio` trên GitHub
2. Upload thư mục này lên repository
3. Vào Settings > Pages > Source: Branch > main
4. Trang sẽ có sẵn tại: `https://yourusername.github.io/emhieu-bio/`

## 📝 Ghi chú

- Tất cả links đã được set `target="_blank"` nên sẽ mở tab mới
- Font Awesome icons được load từ CDN nên cần internet
- Bootstrap CSS được load từ CDN
- Responsive tới 480px (mobile devices)
